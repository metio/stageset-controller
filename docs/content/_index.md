# StageSet Controller

A Flux-compatible controller for **ordered, gated, multi-stage deployments**
of [ExternalArtifact](https://fluxcd.io/) contents — Helm hooks as a
first-class citizen, with per-stage pruning, typed pre/post actions, and
ApplySet-compliant inventory.
