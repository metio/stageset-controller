How StageSet fits with the rest of your delivery stack: react to upstream
producers that render artifacts, compose with the Flux Operator, and drive
StageSets from an AI assistant.

- **[Producer-aware sources](/integrations/producer-aware-sources/)** — point a
  stage at a producer like [JaaS](https://jaas.projects.metio.wtf/) and re-render
  when it republishes.
- **[Flux Operator integration](/integrations/flux-operator/)** — combine StageSet
  with the Flux Operator's `ResourceSet`.
- **[Claude Code skill](/integrations/claude-code/)** — a skill that helps an
  assistant author and operate StageSets.
- **[MCP server](/integrations/mcp-server/)** — an MCP server an agent uses to
  read status and drive reconciliation.
